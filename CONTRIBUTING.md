Thank you for your interest in contributing to Thaumiel Map Editor! Your help is invaluable in making this tool better for everyone. Before submitting a contribution, please review this guide to ensure a smooth process.

## Reporting Bugs

If you encounter a bug:
1. Search existing [issues](https://github.com/Mr-Baguetter/ThaumielMapEditor/issues) to avoid duplicates.
2. Open a new issue with a clear title and detailed description, including:
   - Steps to reproduce
   - Expected vs. actual behavior
   - Screenshots or logs

## Submitting Pull Requests (PRs)

To submit a PR:
1. Create a new branch for your changes
2. Commit your changes with clear, concise messages.
3. Push your branch and open a PR to the `Dev` branch.
4. Include a description of your changes and reference any related issues.
